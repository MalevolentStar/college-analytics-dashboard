# College-Analytics-Dashboard

An interactive Power BI dashboard designed to analyze and explore college data across five Indian states. The project helps users quickly find college details, view geographic locations on a map, and identify the top-performing colleges based on student preference and enrollment.

---

## Project Overview

This Power BI project was developed to simplify college discovery and comparison using interactive dashboards and dynamic filters. It brings together institutional, geographic, district, and contact information into a single visual analytics solution.

The dashboard includes:

- **Main Dashboard** — filters and analyzes colleges by state, district, and college name
- **Location Dashboard** — map-based visualization to locate colleges geographically
- **Top 3 Dashboard** — highlights the top 3 colleges in each state based on student preference and enrollment rate

---

## Problem Statement

Finding specific college information from a large dataset can be time-consuming and difficult. This dashboard was created to solve that problem by making college data easy to search, filter, compare, and visualize in one place.

---

## Data Used

The project uses a dataset containing college-related information such as:

- College name
- State
- District
- Location
- Contact details
- Enrollment-related information
- Student preference data

The data covers colleges across **five states**:

- Himachal Pradesh
- Madhya Pradesh
- Maharashtra
- Punjab
- Tamil Nadu

---

## Key Features

### 1. Main Dashboard
- Analyzes the geographical distribution of colleges across districts in the five states
- Includes dynamic slicers for:
  - College name
  - State
  - District
- Helps users quickly extract and filter specific college details

### 2. Location Dashboard
- Provides interactive map visualization
- Allows users to directly locate and identify college positions geographically
- Makes exploration of college locations more intuitive

### 3. Top 3 Dashboard
- Identifies and displays the top 3 colleges in each state
- Ranks colleges based on:
  - Student enrollment
  - Institutional preference
- Highlights the most prominent educational institutions by state

---

## Insights Derived

This dashboard helps users:

- Find college details quickly without manually searching large datasets
- Compare colleges across states and districts
- Understand the geographic spread of colleges
- Identify top institutions in each state
- Make data-driven educational comparisons with ease
